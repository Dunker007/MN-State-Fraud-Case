## üß© Dashboard Customization

You can personalize the "Paid Leave Watch" dashboard to focus on metrics that matter to you.

### How to Customize
1. Click the **"Customize Dashboard"** button in the top-right corner of the main grid.
2. The grid will enter **Edit Mode**.
3. **Toggle Visibility**: Check/Uncheck widgets in the sidebar list to show or hide them.
4. **Reorder**: Drag widgets to rearrange their position.
5. **Reset**: Click "Reset Layout" to return to the default view.
6. Click **"Save Changes"** to persist your layout (saved to local storage).

### Available Widgets
- **Fund Burn Rate**: Visual gauge of daily payouts vs deposits.
- **Insolvency Countdown**: Monte Carlo projection of fund bankruptcy.
- **Claim Velocity**: Line chart of incoming applications.
- **Fraud Ring Ticker**: Live list of newly detected Phoenix companies.

## üîç Deep Linking

Project CrossCheck supports "Deep Linking". The URL automatically updates as you navigate, allowing you to share exact views.

- **Example**: `https://crosscheck.mn.gov/?tab=paid_leave&time_range=30d`

Sharing this URL with a colleague will load the dashboard with the **Paid Leave** tab active and the time range set to **30 Days**.
