The Licensing Information Lookup system is undergoing maintenance that impacts the ability to view program records with the Substance Use Disorder license type. We apologize for any inconvenience.

Instructions:

Search for information about licensed providers by name, city, county, ZIP code, program type or license number.

Due to the changes that were effective January 1, 2018, with the repeal of Minnesota Rule 31, and the enactment of Minnesota Statutes 245G, you can now locate chemical dependency treatment programs by selecting "Substance Use Disorder" for a complete list of 245G licensed programs.
